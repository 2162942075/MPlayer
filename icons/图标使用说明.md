# 🎨 视频管理应用图标使用指南

## 📁 已提供的文件

- `video-manager-icon.svg` - 详细版图标（256x256）
- `video-manager-icon-simple.svg` - 简化版图标（更适合小尺寸）
- `generate_icons.ps1` - 自动图标生成脚本

## 🛠️ 转换图标的方法

### 方法1：使用在线工具（推荐）

1. **访问在线转换网站**：
   - https://convertio.co/svg-ico/
   - https://cloudconvert.com/svg-to-ico
   - https://www.icoconverter.com/

2. **上传SVG文件**：
   - 选择 `video-manager-icon-simple.svg`（推荐）

3. **设置转换参数**：
   - 输出格式：ICO
   - 尺寸：256x256（或多尺寸）
   - 质量：最高

4. **下载并重命名**：
   - 下载转换后的文件
   - 重命名为 `video-manager.ico`
   - 放入此目录

### 方法2：安装专业工具

1. **安装 Inkscape**：
   ```
   下载地址：https://inkscape.org/release/
   ```

2. **安装 ImageMagick**：
   ```
   下载地址：https://imagemagick.org/script/download.php#windows
   ```

3. **运行生成脚本**：
   ```powershell
   .\generate_icons.ps1
   ```

### 方法3：使用GIMP

1. 安装GIMP（免费）：https://www.gimp.org/
2. 打开SVG文件，设置导入尺寸为256x256
3. 导出为PNG格式
4. 再用在线工具转换PNG为ICO

## 🚀 快速开始

1. **最简单方法**：
   - 上传 `video-manager-icon-simple.svg` 到 https://convertio.co/svg-ico/
   - 下载转换后的ICO文件
   - 重命名为 `video-manager.ico`
   - 放入 `icons` 目录

2. **构建应用**：
   ```bash
   npm run build
   npm run dist
   ```

3. **查看效果**：
   - 生成的安装包会自动使用新图标
   - 桌面快捷方式显示新图标

## 💡 提示

- ICO文件包含多种尺寸，确保各种场景下都清晰
- SVG是矢量格式，可以无损缩放到任意尺寸
- 建议保留SVG源文件，便于后续修改
- 如需修改颜色或样式，编辑SVG文件即可
