## 适合谁

- 需要管理大量本地视频、剧集和分段内容的用户
- 需要把本地磁盘、挂载盘、OpenList 或远程来源统一纳入管理的人
- 需要搜索、收藏、追看、随机抽取内容的重度观影用户
- 需要下载、转码、扫描、补全元数据的媒体整理用户

## 核心能力

### 1. 媒体库管理
- 按分类组织视频和剧集
- 支持批量扫描、文件监控、自动入库
- 支持读取本地 NFO、补齐媒体信息
- 支持海报、缩略图、片名、时长、分辨率、编码等信息展示

### 2. 搜索与筛选
- 全文搜索标题、描述、文件名等信息
- 高级筛选支持分类、标签、类型、画质、时长、评分等条件
- 支持搜索建议、保存搜索与结果高亮

### 3. 播放与历史
- 快速查看最近播放与详细历史记录
- 支持从列表、收藏、历史、搜索结果中直接播放

### 4. 收藏与标签
- 支持视频和剧集收藏
- 支持标签管理、批量标签操作、标签筛选
- 便于把内容按兴趣、题材或自定义规则二次整理

### 5. 下载与转码
- 内置通用下载管理
- 支持 BT / 磁力相关任务管理
- 支持网页视频下载
- 支持视频格式转换与并发任务管理

### 6. 远程来源与索引
- 支持来源索引与局部同步
- 适配 OpenList 等挂载来源
- 可查看索引文件、目录、任务历史和引用关系
- 适合统一管理本地与远程媒体来源

### 7. 随机观影
- 提供随机抽取内容的“盲盒”入口
- 可直接播放单个视频，或进入剧集详情继续选择
- 适合快速开看，不用先翻库



## 产品亮点

- 桌面端本地应用，数据可控
- 覆盖从入库、整理、搜索到播放的完整链路
- 同时兼顾普通本地库和挂载来源场景
- 适合长期积累和维护个人媒体资产





## 一句话介绍

PlayerMacTwo 不是单纯的播放器，而是一套把媒体库整理、检索、追看、下载和来源管理串起来的本地视频管理工具。
